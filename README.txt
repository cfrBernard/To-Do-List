Classic To-Do List

A simple web application that allows users to manage their tasks and to-do lists directly in their browser. The app uses HTML, CSS, and JavaScript, with data stored locally via localStorage to enhance the user experience. Users can add and delete tasks, and all data is saved in their browser, ensuring privacy and easy access.

Features:

Create and delete tasks.
Data persistence with localStorage.
Responsive design for mobile and desktop use.
No external dependencies – pure HTML, CSS, and JavaScript.
